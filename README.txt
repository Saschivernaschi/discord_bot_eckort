Entwicklung von DiscordBot Eckort.
Der Bot wird hauptsächlich in Java programmiert und sollte folgende Funktionen zur Verfügung stellen:

- Minispiele
    - Kumpelquiz aus WII-Party 
        - 1 Frage -> 4 dazugehörige Antworten
        - eheste Antwort wird ausgewählt
        - Antworten werden mit Button ausgewählt
        - Statistik + Tags für die besten Freunde (Winning Lines)
        - Datenbank mit Fragen
            - Fragen und Antworten können von Nutzer angelegt werden
    - Mini-Rennspiel (Rennspiel)
        - arrows als reaction einbauen, um das Auto zu steuern
- Sprachmodell einbauen
- Transkripte erstellen lassen
- Spielstatistiken von League anzeigen lassen
- Bot-Statistiken: Zeige Statistiken und Nutzungsdaten des Bots an, wie z.B. Anzahl der gesendeten Nachrichten, häufig verwendete Befehle, etc.
- Zitate und Weisheiten: Poste regelmäßig inspirierende Zitate, Witze oder Weisheiten.
    - Disskusionsthemen
    - AI einbauen
- Feedback-System: Erlaube Benutzern, anonymes oder offenes Feedback zu geben, um den Server oder den Bot zu verbessern.
- Steuerung für den vrisingserver
